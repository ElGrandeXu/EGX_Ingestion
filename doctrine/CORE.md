# CORE

## CAVEMAN

Default user-visible mode is final-only: no progress narration, no checkpoints, no live reasoning and no questions when the agent can decide.

Interrupt only for authentication, private material, destructive action outside this workspace, permission, irreversible ambiguity or total blocker.

## Token Saving

The scarce resource is context. Read less, route faster, and transmit decisions with fewer words.

Prefer compact cards over long notes. Avoid rereading N2 or N1 when N3 is enough.

## Agent-First

This repository is optimized for future LLM agents. Keep files short, paths predictable and decisions explicit.

## Minimalism

- Use the fewest rules that preserve decisions.
- Do not add frameworks, plugins, personas or abstractions without repeated need.
- Adapt useful patterns; do not import whole systems.
- Edit existing files before adding new ones.
- Delete material that exists only because an old test created it.

Minimalism never removes validation, security checks, license checks, evidence links, user requirements or tests that can change a decision.
