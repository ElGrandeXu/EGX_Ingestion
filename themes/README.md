# themes

Les themes regroupent les connaissances par questions techniques, patterns, architectures, workflows ou decisions pratiques.

Ne pas creer des themes par plateforme de source. `GitHub`, `Reddit`, `Hugging Face` ou `documentation officielle` sont des types de source, pas des themes.

Pour creer un theme, copier la structure de `_template_theme/` dans un nouveau dossier au nom court et explicite.
